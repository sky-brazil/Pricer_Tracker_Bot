## Checklist de revisão manual (para você validar no seu perfil)

1. Headline comunica **nicho + resultado + stack**?
2. Overview mostra **dor -> solução -> prova -> CTA**?
3. Existem **3 itens de portfólio** com prova visual?
4. Há **pacotes fechados** (Basic / Standard / Premium)?
5. Texto está em inglês claro e sem excesso de termos genéricos?
6. Cada item mostra métricas (tempo economizado, throughput, etc.)?

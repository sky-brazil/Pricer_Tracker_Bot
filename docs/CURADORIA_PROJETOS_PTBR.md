# Curadoria dos 3 Projetos (PT-BR)

Este documento organiza os 3 projetos para uso comercial no Upwork com posicionamento claro.

## 1) Lead Enrichment Automation

Pasta: `portfolio-projects/01-lead-enrichment-automation`

### Dor do cliente
Leads em planilhas bagunçadas, sem prioridade clara de abordagem.

### Entrega do projeto
- normalização dos dados;
- enriquecimento básico (domínio);
- score de lead;
- export pronto para outreach.

### Oferta Upwork recomendada
- **Basic:** limpeza e normalização de CSV;
- **Standard:** enriquecimento + score + export filtrado;
- **Premium:** pipeline recorrente + alerta.

## 2) Price Monitoring API

Pasta: `portfolio-projects/02-price-monitoring-api`

### Dor do cliente
Monitoramento manual de preços é lento e pouco confiável.

### Entrega do projeto
- endpoint de health;
- endpoint de checagem de preços;
- retorno estruturado com alertas;
- base pronta para scheduler e integrações.

### Oferta Upwork recomendada
- **Basic:** monitoramento de 1 site + alerta simples;
- **Standard:** múltiplas fontes + API;
- **Premium:** API + painel + deploy.

## 3) Job Intelligence Dashboard

Pasta: `portfolio-projects/03-job-intelligence-dashboard`

### Dor do cliente
Dados de vagas dispersos sem visão executiva para decisão.

### Entrega do projeto
- relatório Excel com múltiplas abas;
- agregação por empresa e localização;
- resumo executivo em Markdown.

### Oferta Upwork recomendada
- **Basic:** limpeza e agregação de dataset;
- **Standard:** relatório recorrente;
- **Premium:** pipeline completo + dashboard.

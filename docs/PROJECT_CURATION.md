# Curation Guide for the 3 Portfolio Projects

This guide positions each project for commercial use on Upwork.

## 1) Lead Enrichment Automation

Path: `portfolio-projects/01-lead-enrichment-automation`

### Client pain
Lead spreadsheets are inconsistent and hard to prioritize.

### Delivery
- data normalization;
- basic enrichment (domain extraction);
- lead scoring;
- outreach-ready export.

### Recommended Upwork offer
- **Basic:** CSV cleanup and normalization;
- **Standard:** enrichment + scoring + filtered export;
- **Premium:** scheduled pipeline + notification.

## 2) Price Monitoring API

Path: `portfolio-projects/02-price-monitoring-api`

### Client pain
Manual competitor price checks are slow and unreliable.

### Delivery
- health endpoint;
- one-run price check endpoint;
- structured alert payload;
- easy path to scheduling/integration.

### Recommended Upwork offer
- **Basic:** one source + one alert channel;
- **Standard:** multi-source monitoring + API;
- **Premium:** API + dashboard + deployment.

## 3) Job Intelligence Dashboard

Path: `portfolio-projects/03-job-intelligence-dashboard`

### Client pain
Raw jobs datasets do not provide clear executive insights.

### Delivery
- multi-sheet Excel report;
- aggregation by company and location;
- executive markdown summary.

### Recommended Upwork offer
- **Basic:** one dataset cleanup + aggregation;
- **Standard:** recurring report generation;
- **Premium:** full pipeline + dashboard + delivery automation.

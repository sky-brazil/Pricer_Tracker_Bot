# Documentation Index

This folder contains client-facing and profile-facing artifacts prepared for
international hiring, freelance marketplaces, and business development.

## Files

- `UPWORK_PROFILE_KIT_EN.md` - reusable headline, summary, packages, and
  proposal language for Upwork and similar platforms.
- `GITHUB_BIO_OPTIONS.md` - short, professional GitHub bio options ready to
  copy and paste.
- `GITHUB_PROFILE_README.md` - complete profile README template for the
  `sky-brazil/sky-brazil` repository.
- `INTERNATIONAL_READINESS_CHECKLIST.md` - pre-share checklist for GitHub and
  profile quality validation.
